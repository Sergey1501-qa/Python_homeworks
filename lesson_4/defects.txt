БАГ-РЕПОРТЫ ДЛЯ StringUtils
============================

ДЕФЕКТ #1: Метод trim() не удаляет табуляции и другие пробельные символы
Описание: Метод удаляет только обычные пробелы (" "), но не удаляет:
          - Табуляции (\t)
          - Переносы строк (\n)
          - Возвраты каретки (\r)
Воспроизведение:
  utils.trim("\tskypro")  # Возвращает "\tskypro" вместо "skypro"
Ожидаемое поведение: Удаление всех пробельных символов в начале строки
Серьезность: Средняя

ДЕФЕКТ #2: Метод contains() неэффективен
Описание: Используется try/except для контроля потока вместо простой проверки
Рекомендация: Заменить на `return symbol in string`
Серьезность: Низкая

ДЕФЕКТ #3: Метод delete_symbol() делает лишнюю проверку
Описание: Сначала проверяется contains(), потом вызывается replace()
          Метод str.replace() уже возвращает исходную строку при отсутствии символа
Рекомендация: Упростить до `return string.replace(symbol, "")`
Серьезность: Низкая
