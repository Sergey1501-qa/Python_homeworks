# Проект автоматизации тестирования

## Описание проекта
Автоматизация тестирования веб-приложений с использованием Selenium WebDriver и Allure Reports.

## Тестируемые приложения
1. **Медленный калькулятор** - проверка работы калькулятора с задержкой
2. **Sauce Demo** - интернет-магазин для тестирования полного цикла покупки

## Технологический стек
- Python 3.11+
- Pytest
- Selenium WebDriver
- Allure Reports
- PageObject паттерн
